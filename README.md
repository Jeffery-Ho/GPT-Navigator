# Polaris for Web

一个给 AI 网页回复增加右侧段落导航的浏览器扩展。

支持 ChatGPT、豆包、Kimi、千问、元宝。插件会自动识别 assistant 回复里的标题，在右侧生成 marker，点击即可跳转到对应段落。

## 主要功能

- 自动识别 H1/H2/H3 标题，并按平台规则适配。
- 右侧 marker 支持点击跳转、当前选中高亮、滚动列表。
- 设置中可按当前平台过滤 Marker 类型：H1 / H2 / H3。
- 设置会通过 `chrome.storage.sync` 同步。

## 安装

1. 打开浏览器扩展管理页面。
2. 开启开发者模式。
3. 选择“加载已解压的扩展程序”。
4. 选择本项目文件夹。
5. 刷新 AI 网页。
